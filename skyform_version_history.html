V6 Weekly Management Report & HSEQ Manager April 2026

Adds the Weekly Management Report feature and the HSEQ Manager role. The report is generated automatically from live system data and covers the previous 7 days of issues plus the next 7 days of risks, designed specifically for the Monday management meeting.

Weekly Management Report — auto-generated from live data, covering critical compliance, expiring certs, unauthorised absences, overdue workscopes, audit docs, amber-risk engineers for next week and upcoming workscope starts

+
Print / Save PDF — report opens in a modal with a print button for distribution before management meetings
▲

V5 Interactive Planner — Day-Grid Model April 2026

Complete rebuild of the Interactive Planner using a day-level data model. Engineers are allocated to specific calendar dates rather than week slots, enabling precise clash detection and accurate absence integration. Seeded with five live programme teams.

+
Day-grid planner — 2/3/4-week scrollable calendar with per-day engineer allocation and week-boundary grouping
+
Live programme seed data — 5 teams pre-loaded: Hornsea 1 (KA+JB), Beatrice (BM+BA), Whitelee (MA+TK, 3 wks), Blacklaw (PC+RA+GM, 3-man), Bhlaraidh (MD+MB, starts w/b 4 May)
+
Engineer picker drawer — tap any day cell or select a date range then tap + Add Engineer; picker shows real-time compliance status and absence warnings
+
Day detail modal — tap any allocated cell to see all engineers for that day with option to add or remove
+
Compliance chip colours — green/amber/red chips in the grid; tap any chip to open the full compliance panel at the top
▲
Full iOS Safari compatibility — all toLocaleDateString, .closest(), optional chaining, inset:0 and other iOS incompatibilities removed
●
Export Plan — saves current allocation as a formatted text file
